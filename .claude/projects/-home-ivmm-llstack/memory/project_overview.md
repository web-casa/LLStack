LLStack is a CLI + TUI Web Stack Installer & Site Lifecycle Manager for EL9/EL10 (Rocky/AlmaLinux/RHEL).

**Why:** Not a web panel or bash script - it's a plan/apply/verify/rollback lifecycle tool for Apache/OLS/LSWS backends.

**How to apply:** All work builds on existing canonical architecture. Never re-do completed phases. Key entry points in dev_docs/HANDOFF_TO_CLAUDE_CODE.md.
